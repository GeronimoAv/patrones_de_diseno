# patrones de diseño

## definicion

son soluciones habituales a problemas que se presentan concurrentemente en la industria, no son partes del codigo, si no conceptos completos para la solucion de problemas.

## clasificacion de los patrones de diseño

### patrones creacionales
se centran en la creacion de objetos, incrementando la flexibilidad y reutilizando codigo existente.

## patrones estructurales
